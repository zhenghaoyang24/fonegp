## 🚀 核心功能

### 🗓️ 实时赛程
- 完整详细的比赛日程

### 📊 赛后分析
- 互动式单圈时间对比
- 轮胎策略

### 🏆 排名系统
- 含状态指标的车手积分榜
- 车队冠军进度追踪

### 🔍 深度档案
- 车手职业生
- 车队历史
